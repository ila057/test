Help tag data, train algorithms & make this world a better place

What is a bounding box?


Bounding boxes are n-sided polygons which encircle a particular object very accurately.


These boxes can be used to train the machienes to recognise the objects present inside the box. Giving the computer machiene a lot of examples can help it understand the features and properties of the product and then identify the product automatically the next time.


How to draw perfect bounding boxes?


Accuracy in box size: The box should be as tight as possible to the object which is to be bounded.
It should NOT intersect/ cut the object at any place.

On a whole it should neither intersect the object and nor be too loose.

Precise Labels: The box should be correctly categorised into one of the given labels.

Drawing a box with incorrect label will feed in the wrong information into the computer and the system will not be able to identify the products correctly from next time.

Task types:


Two types of bounding box tasks exist. Number of boxes per image can change depending on the image.
